FASTEST WAY (GitHub + Vercel):
  1. git init && git add . && git commit -m "Initial"
  2. Push to GitHub
  3. Go to vercel.com → Import Repository
  4. Click Deploy
  5. Done! Live in ~30 seconds

ALTERNATIVE (Vercel CLI):
  1. npm i -g vercel
  2. vercel
  3. Follow prompts
