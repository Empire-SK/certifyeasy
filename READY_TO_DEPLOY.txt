📊 CSV FORMAT REQUIRED
══════════════════════

name,eventName,certificateId,issueDate
John Doe,AI Workshop,CERT-001,2024-04-24
Jane Smith,Web Development,CERT-002,2024-04-24

Columns (must match exactly):
  ✓ name - Full name of recipient
  ✓ eventName - Name of event/course
  ✓ certificateId - Unique identifier (no duplicates)
  ✓ issueDate - Date in YYYY-MM-DD format

═══════════════════════════════════════════════════════════════════════════════

🔌 API ENDPOINTS
════════════════

1. POST /api/certificates
   Issue a single certificate
   
   Input: { name, eventName, certificateId, issueDate }
   Returns: { success, message, certificate }

2. POST /api/upload-certificates
   Upload multiple certificates via CSV file
   
   Input: multipart/form-data with "file" field
   Returns: { success, message, count }
